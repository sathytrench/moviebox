## Installation

OS X & Linux:

```sh
`npm install`
`createdb moviebox`
`npm run seed`
`npm run start-dev`
Find the app at http://localhost:3000. 
```

Windows:

```sh
`npm install`
`createdb moviebox`
`npm run seed`
`npm run build-client-watch`, then `npm run start-server`
Find the app at http://localhost:3000. 

```
